In this tutorial, you will learn to make a table with columns and rows.

You must understand the <th>,<td> ,<table> and <tr> tags.

The <table> tag is the base tag for a table. Everything inside the table is sorrounded within this tag and its closing tag :</table>

The <th> refers to a table heading this is used to give headings to a table and determines how many columns you have. The more <th> means more columns. <th> also has a closing tag which is:</th>

The <td> tag refers to table data these are usually sorrounded by the <tr> tags because this is the "data" that goes in the columns / rows. <td> also has a closing tag which is: </td>

The <tr> tag also has a closing tag which is: </tr> This tag refers to a table row. The more of these tags means more rows in the table. These tags sorround the <td> tags.
