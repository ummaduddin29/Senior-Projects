Although HTML is one of the easiest computer programming languages to learn, it can become very hard very quickly. That is why some rules should be memorized. These are some rules that apply to HTMl.

1. The <DOCTYPE html> Tag/Declaration Is Required.

This determines what type of document you are using. This declaration helps browsers display you webpage correctly. This command should also only appear once through your code, and it should always be located at the very top. You might have already seen this on the Home page. The HTML document itself begins with <html> and ends with </html>.

2. Not All HTML Tags Have A Closing Tag

As you might have already seen on the Basic HTML Tags List Page, there are some tags that do not have a closing tag. These tags are called empty tags because they dont have anything in them which requires them to be closed.

Examples:
  • <br>
  • <hr>

The tags that do require a closing tag are the same tag but with a forward slash after the first angle bracket.

Examples:
  • <body> and </body>
  • <center> and </center>

3. All Visible Content Goes In The <body> Tag

Most of the content such as headings and text or images that you want to show is included in the between the <body> and the </body> sections. The <body> is not an empty tag so it has a closing tag.This is the same way you should be enclosing other tags such as <html>, <center> and <em>

In the example above, the content (the heading and paragraph) are contained by the <body> and the </body> tags and will be the only parts shown on the real website.

4. Don't Forget To Close Non-Empty Tags

This should be an obvious, DON'T FORGET TO CLOSE YOUR TAGS. If you don't, the HTML will not show up or will mess up the rest of the website. Opening and Closing tags work like an on and off switch, the opening tag turns it on and the closing tag turns it off.

5. Don't Forget The Order, Order, Order!

If you start a tag inside another tag, that tag must be closed before the tag thats outside that tag, this can confuse old browsers but has been allowed in newer versions. So its to your benefit and increases usability if you stick to this rule.
