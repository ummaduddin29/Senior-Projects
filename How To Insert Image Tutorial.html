In this brief tutorial, you will learn to insert an image.

As you might have already seen on the Basic HTML Tags page, to insert an image we need to use the <img> tag.

This is what the image tag looks like when its used. You can later go in and add width="x-width" and a height="y-height" elements before the closing brace (>) to resize your image. You can also add some alternative text for visually impaired persons and a screen reader will read it out to the person with a alt="alternative text here" element.
