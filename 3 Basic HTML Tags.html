    Basic HTML Tags List



NOTE:

Some of the following tags may not have a closing tag because they simply do not need one.

Opening Tag Closing Tag Description
<DOCTYPE html> ----- Specifies what type of document you are using.
<html> </html> Specifies that you are using an HTML document.
<head> </head> Information about a document.
<style> </style> Used to introduce CSS into your code.
<body> </body> Defines the body (main part) of the HTML code.
<h1> to <h6> </h1> to </h6> Html heading to a page , <h1> is the largest in size.
<p> </p> Opens a paragraph which you can write in.
<br> ----- Skips a line on the actual website page.
<hr> ----- Makes a "Horizontal Rule" or a straight line.
<center> </center> Centers all text it surrounds.
<em> </em> Puts an "emphasis" or bolds anything it surrounds.
<i> </i> Puts text in italics.
<strong> </strong> Makes your text "stronger" / bolds it.
<img> ----- Defines / inserts an image.
<a> </a> Makes a link.
<ul> </ul> Makes an unordered List.
<ol> </ol> Makes an ordered list.
<table> </table> Makes an table.
<th> </th> Makes a table heading --- used with a <table> tag.
<td> </td> Inserts table data --- used with a <table> tag.
<tr> </tr> Makes a table row --- used with a <table> tag.
<div> </div> Inserts a box / division
